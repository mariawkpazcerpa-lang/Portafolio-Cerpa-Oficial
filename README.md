Portafolio web personal desarrollado para presentar mis proyectos, habilidades y experiencia como Frontend Developer.

## Tecnologías

- HTML5
- SCSS / Sass
- JavaScript
- Swiper.js
- npm

## Características

- Diseño responsive
- Tema claro y oscuro
- Galería de proyectos dinámica
- Formulario de contacto
- Navegación adaptable a dispositivos móviles
